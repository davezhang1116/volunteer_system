## Volunteer & Research Matching System
This is a potential addition for the [American Data]("https://www.american-data.com/")’s volunteering system.
## Objective
To develop a dedicated volunteer scheduling and matching application  for the American Data ecosystem. This system goes beyond manual shift tracking by utilizing a matching engine to pair volunteers with clinical researchers, care coordinators, and long-term care residents based on skills, research interests, and semantic compatibility

Please refer to the document for more information.
## How to run the application
1. Install the dependencies: `pip install -r requirements.txt`
2. Initialize the database with some random data: `python seed.py`
3. Run the application: `python run.py`
